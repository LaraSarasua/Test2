# Ensolvers Notes App
SPA for note management with Node.js, Express, Sequelize and MySQL.

## Technologies
- Node.js v18+
- Express
- Sequelize + mysql2
- MySQL 8+
- Vanilla JS, HTML y CSS

## How to execute
1. Clone repo  
2. Configure `backend/.env`.  
3. Create DB:
    ```sql
   CREATE DATABASE ensolvers_notes;
